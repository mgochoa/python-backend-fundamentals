- Need a quick schema lookup

## üéØ How to Use These Docs

### For Beginners (Stories 1-3)
1. Start with [DATABASE_SCHEMAS.md](DATABASE_SCHEMAS.md)
2. Study the Library System diagram
3. Read the relationship explanations
4. Try the example queries in `playground.py`

### For Guided Work (Stories 4-9)
1. Reference [SCHEMA_QUICK_REFERENCE.md](SCHEMA_QUICK_REFERENCE.md)
2. Look at the Todo System diagram
3. Use it while completing TODOs
4. Check query examples when writing your own

### For Challenge Work (Stories 10-15)
1. Study all three schemas in [DATABASE_SCHEMAS.md](DATABASE_SCHEMAS.md)
2. Compare patterns across schemas
3. Use the design checklist
4. Reference when implementing your inventory system

## üîç Viewing Diagrams

### On GitHub
The Mermaid diagrams in `DATABASE_SCHEMAS.md` render automatically on GitHub. Just view the file on GitHub.com to see the visual diagrams.

### In VS Code
Install the "Markdown Preview Mermaid Support" extension to view diagrams in VS Code.

### In Other Editors
- Copy the Mermaid code to [Mermaid Live Editor](https://mermaid.live/)
- Or use the ASCII diagrams in `SCHEMA_QUICK_REFERENCE.md`
