# Test Files

This directory contains test files for validating the learning project implementation.

## Test Files

- `test_validators.py` - Tests for validation functions
- `test_error_handlers.py` - Tests for error handling utilities
- `test_library_models.py` - Tests for the library system (reference implementation)
- `test_task_11_1.py` - Integration tests for the complete project

## Running Tests

Run all tests:
```bash
pytest tests/
```

Run a specific test file:
```bash
pytest tests/test_validators.py
```

Run with verbose output:
```bash
pytest tests/ -v
```

## Note

These tests are provided as examples and for validation. Students can use them to verify their implementations are working correctly.
